Knowledge framework of energy storage systems for vehicles using mind mapping

## 项目初衷

- 建立系统化的电动车辆复合储能系统知识体系；
- 分享复合储能系统及能量管理控制相关研究论文；
- 相关系统模型和控制算法的开源代码。
